## 🐛 BUG 修復

### preview.html 導向錯誤的模板檔案
- 問題：所有版面都開啟 `display.html`（不存在的檔案）
- 原因：舊版程式碼寫死檔案名稱
- 修復：動態判斷模板類型
- 測試：
  - default 版面 → `default.html` ✅
  - 雙影片測試版面 → `dual_video.html` ✅

### 瀏覽器快取導致舊版本持續運作
- 問題：部署新版本後，瀏覽器仍執行舊的 JavaScript
- 發現：主控台顯示 `PLAYER_VERSION: 5.3.3` 而非 `5.3.5`
- 解決方案：
  - 提醒使用者執行硬重新整理：`Ctrl+Shift+R`
  - 或使用無痕模式測試
  - 或清除瀏覽器快取
- Cloudflare Workers 部署後需要時間同步到邊緣節點
- 等待 1-2 分鐘後重新測試確認版本更新

### node_modules 殘留測試套件
- 問題：刪除 package.json 依賴後，node_modules 仍有舊套件
- 執行：`npm install` 或 `npm prune`
- 結果：成功移除 73 個不需要的 package
- 專案體積減少：約 15MB
