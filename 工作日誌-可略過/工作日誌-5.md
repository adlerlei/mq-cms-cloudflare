### Admin 導航欄 Debug 按鈕
- 位置：navbar-end 區域，登出按鈕左側
- HTML 結構：
  ```html
  <a href="/preview.html" target="_blank" class="button is-info is-light">
  ```

- 圖標：Font Awesome `fa-bug` icon
- 功能：點擊開啟 preview.html 於新分頁

### preview.html 自動啟用調試模式
- 所有預覽鏈接自動附加 `&debug=true` 參數
- 修改按鈕文字：
  - 從：「預覽 {layout.name} 版面」
  - 改為：「預覽 {layout.name} 版面 (調試模式)」
- 更換圖標：
  - 從：`<i class="fas fa-play"></i>`
  - 改為：`<i class="fas fa-bug"></i>`
- JavaScript 程式碼：
  ```javascript
  container.innerHTML = layouts.map(layout => {
      const templateFile = layout.template === 'dual_video' 
          ? 'dual_video.html' : 'default.html';
      return `<a href="/${templateFile}?deviceId=preview-${layout.name}&debug=true" ...>`;
  }).join('');
  ```
- 修改檔案：`public/preview.html` (loadLayouts function)

## 🛠️ 更新與改進

### 移除所有鍵盤快捷鍵功能
- 刪除 keydown event listener（約 40 行）
- 移除的快捷鍵：
  - `Ctrl+Shift+D` / `Cmd+Shift+D` - 切換調試模式
  - `Ctrl+D` / `Cmd+D` - 切換調試面板
  - `Ctrl+R` / `Cmd+R` - 強制重新載入
  - `Ctrl+V` / `Cmd+V` - 檢查版本
- 移除原因：
  - Chrome 瀏覽器：`Ctrl+Shift+D` 會開啟書籤管理器
  - Safari 瀏覽器：部分快捷鍵無反應
  - Firefox 瀏覽器：`Ctrl+D` 會觸發書籤功能
  - 跨瀏覽器兼容性差，使用者體驗不佳
- 保留功能：URL 參數控制 (`?debug=true`)
- 修改檔案：`public/js/animation.js`
- 替換為註解：`// Keyboard shortcuts removed - debug mode is now controlled via URL parameter only`

### 調試提示訊息優化
- showDebugInfo() function 中的提示更新
- 舊訊息：

  ```javascript
  debugLog('💡 Tip: Press Ctrl+Shift+D to toggle debug mode, Ctrl+D to toggle debug overlay');
  ```

- 新訊息：

- 更符合實際使用方式（透過 Debug 按鈕）
